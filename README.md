# Football players - React App

A simple React application that displays a list of football players and allows users to filter and sort them by various criteria.  
Built as a practice project to improve React, SASS, and UI/UX structuring skills.

## Technologies

* HTML
* SCSS
* Javascript
* React
* Font Awesome
* Vite
* ESLint
